# Jupiter API - Developer Experience Report

This is a hand written developer experience report for [https://developers.jup.ag/](https://developers.jup.ag/)

## Onboarding

Onboarding process was very smooth, but only after I decided to use Jupiter API for my project. Reading the docs was convoluted, and it took me a while to understand what is being offered here and decide if it is worth integrating.

I gave claude just the link to the website and it figured out how it integrate with the docs by itself.

## What's broken or missing in the docs?

The prediction market docs are confusing. It gave me an impression I can create my own markets as that is generally what apis are used for, rather than interacting with existing markets on different sites.

It is not clear to me why should I integrate this in my project, what will I be able to do which I can't already do without it or with using other libraries.

Also no light mode on developer portal.

### Where did the APIs bite you?

Didn't face issues with API as it was managed by Ai Agent. Prediction market apis gave me impression that I can create prediction markets, I dont know why.

### Did you use the AI stack?

Worked great. No complaints.

### How would you rebuild developers.jup.ag?

I will reposition this as a wallet or atleast integrate with a existing api wallet.

Please checkout [zerion-ai](https://github.com/zeriontech/zerion-ai).

The wallet is very powerful and can be used directly while creating agents and other automated applications. But at the same time missing a few features that this API offeres.

Please looking into a possible integration as it will solve many of the problems mentioned above as well as provide a 10x more value to developers.

### What do you wish existed?

I wanted Jupiter API to also provide tools for new defi users to get started with defi and using Jupiter terminal in a easy way.

When I mean tools, I do not mean tools on the platform but specific API endpoints that help navigate and use the trading terminal as it is pretty intimidating for people from non trading backgrounds.
